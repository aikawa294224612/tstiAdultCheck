Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

成人預防保健-健康行為

此Profile用於紀錄成人預防保健中健康行為區塊資訊，包含喝酒、嚼檳榔、運動，與曾於成人預防保健服務時接受過 B、C 型肝炎檢查之項目。(特別注意: 不包含吸菸，吸菸將直接使用臺灣核心-吸菸狀態（TW Core Smoking Status Observation）)
成人預防保健-健康行為-吸菸

此Profile繼承於臺灣核心-吸菸狀態（TW Core Smoking Status Observation），用於紀錄成人預防保健中健康行為區塊資訊之吸菸狀況。
成人預防保健-健康諮詢

此Profile用於紀錄成人預防保健中健康諮詢區塊資訊，包含健康諮詢：戒菸、健康諮詢：節酒、健康諮詢：戒檳榔、健康諮詢：規律運動、健康諮詢：維持正常體重、健康諮詢：健康飲食、健康諮詢：事故傷害預防、健康諮詢：口腔保健、健康諮詢：腎病識能衛教指導、健康諮詢：慢性疾病風險評估(1140101新增)。
成人預防保健-受檢民眾基本資料

此Profile繼承於臺灣核心-病人(TW Core Patient)，用於紀錄成人預防保健服務檢查紀錄受檢民眾資訊。
成人預防保健-受檢民眾疾病史

此Profile繼承於臺灣核心-病情、問題或診斷（TW Core Condition），用於紀錄成人預防保健服務檢查紀錄疾病史資訊，包含高血壓、糖尿病、高血脂症、心臟病、腦中風、腎臟病。
成人預防保健-實驗室檢查

此Profile繼承於臺灣核心-實驗室檢驗檢查（TW Core Observation Laboratory Result），用於紀錄成人預防保健實驗室檢查區塊，包含尿液蛋白質、總膽固醇、三酸甘油脂、低密度脂蛋白膽固醇、高密度脂蛋白膽固醇、GOT、GPT、肌酸酐、腎絲球過濾率（eGFR）、B 型肝炎表面抗原（HBsAg）、C 型肝炎抗體（Anti-HCV）、尿酸(114.1.1 更新)。
成人預防保健-就醫紀錄

此Profile繼承於臺灣核心-就醫事件（TW Core Encounter），用於紀錄成人預防保健服務檢查紀錄第一階段檢查日期與第二階段檢查日期。
成人預防保健-慢性疾病風險

此Profile用於紀錄成人預防保健慢性疾病風險資訊(114.1.1新增)。
成人預防保健-憂鬱檢測

此Profile用於紀錄成人預防保健服務檢查紀錄憂鬱檢測
成人預防保健-服務診治醫師

此Profile繼承於臺灣核心-健康照護服務提供者（TW Core Practitioner），用於紀錄成人預防保健兩階段服務診治醫師代號資訊。
成人預防保健-檢查結果與建議

此Profile繼承於臺灣核心-診斷報告（TW Core DiagnosticReport），用於紀錄成人預防保健檢查結果與建議資訊
成人預防保健-腎功能檢查期別

此Profile用於紀錄成人預防保健腎功能檢查期別(1140101新增)。
成人預防保健-身體檢查-腰圍

此Profile用於紀錄成人預防保健中腰圍資訊。
成人預防保健-身體檢查-血壓

此Profile繼承於臺灣核心-血壓（TW Core Observation Blood Pressure），用於紀錄成人預防保健中血壓(舒張壓、收縮壓)資訊。
成人預防保健-身體檢查-身體質量指數(BMI)

此Profile繼承於臺灣核心-身體質量指數（TW Core Observation BMI） ，用於紀錄成人預防保健中身體質量指數(BMI)資訊。
成人預防保健-身體檢查-身高

此Profile繼承於臺灣核心-身高（TW Core Observation Body Height），用於紀錄成人預防保健中身高資訊。
成人預防保健-身體檢查-體重

此Profile繼承於臺灣核心-體重（TW Core Observation Body Weight） ，用於紀錄成人預防保健中體重資訊。
成人預防保健-醫事機構資訊

此Profile繼承於臺灣核心-醫事機構（TW Core Organization Hospital Department），用於紀錄成人預防保健服務檢查紀錄醫事機構與委託代檢醫事檢驗機構之機構資訊。
